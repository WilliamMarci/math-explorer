# MathMap

MathMap 是一个基于 **React** 和 **D3.js** 构建的交互式数学概念图谱应用，支持可视化编辑和管理数学知识结构。应用注重流畅的交互体验和现代化界面设计，适合数学学习、知识梳理和学科图谱构建。

## 功能特性

### 1. 图谱画布与交互

- **力导向布局**：基于 D3.js 实现节点斥力、连线拉力和向心力的物理模拟。
- **无限画布**：支持缩放（滚轮）和平移（拖拽）。
- **节点拖拽与固定**：节点可拖动，释放后自动固定（Pin）。
- **小地图**：右上角显示全局视图，支持调整大小，实时反映视口范围。

### 2. 节点系统

- **多类型节点**：支持 Concept、Axiom、Constant、Parameter、Note 等多种数学概念类型，拥有独特样式。
- **节点状态**：
    - **Hover**：悬停时节点上浮并显示操作按钮。
    - **Pinned**：固定节点显示发光图钉图标。
    - **Focus**：双击或列表选择节点，自动居中定位。
- **节点操作**：
    - **新建**：通过分裂按钮快速创建不同类型节点。
    - **编辑**：双击节点弹窗编辑，支持 LaTeX 数学公式。
    - **右键菜单**：支持 Focus、Edit、Pin/Unpin、Hide、Delete 等操作。

### 3. UI 界面与布局

- **资源管理器（左上角）**：
    - 折叠/展开面板，内含搜索、节点列表、分裂按钮。
    - 节点列表支持跳转、右键菜单、固定状态显示。
- **设置面板（左下角）**：
    - 底部标题栏，点击展开/收起。
    - 个性化设置：语言切换（中/英）、主题切换（多配色）。
    - 物理引擎调优：滑块实时调整重力、向心力、连线距离。
    - 显示开关：控制 Tooltips、小地图显示。
    - 数据管理：导入/导出图谱数据（JSON/MathMap 格式）。

### 4. 视觉与动画

- **平滑动画**：面板展开/收起采用贝塞尔曲线，动画自然。
- **内容淡入淡出**：避免视觉闪烁，提升体验。
- **现代化风格**：基于 Tailwind CSS，圆角、阴影、磨砂玻璃、RemixIcon 图标。

## 总结

MathMap 已实现完善的思维导图/知识图谱编辑功能，具备物理引擎驱动的流畅交互、直观的 UI 布局和丰富的节点管理能力。适用于数学知识梳理、学科图谱构建等多种场景。
